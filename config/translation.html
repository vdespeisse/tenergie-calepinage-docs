# translation.json

  • Type: { [key: String]: [translation: String] }

Clés de traduction des variables du modules.

Ex:

{
  "south": "Sud"
}

"south" sera affiché comme "Sud" dans l'interface.

A priori vous n'avez pas a modifier ce fichier.
